# Markdown 常用语法

## 1. 标题

```markdown
# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题
```

## 2. 字体

- **粗体**：`**粗体**` 或 `__粗体__`
- _斜体_：`*斜体*` 或 `_斜体_`
- **_粗斜体_**：`***粗斜体***` 或 `___粗斜体___`

## 3. 分隔线

---

```markdown
---
---

---
```

## 4. 删除线

- ~~删除线~~：`~~删除线~~`

## 5. 列表

### 无序列表

```markdown
- 第一项
- 第二项
- 第三项
```

### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 6. 区块引用

```markdown
> 这是一个区块引用

> > 嵌套区块
```

## 7. 代码

````markdown
```javascript
console.log('111')
```
````

## 8. 链接

```markdown
[链接名称](链接地址)

<链接地址>
```

## 9. 图片

```markdown
![alt 属性文本](图片地址 '可选标题')
```

## 10. 表格

```markdown
| 表头   | 表头   |
| ------ | ------ |
| 单元格 | 单元格 |
| 单元格 | 单元格 |
```

设置对齐方式：

- `:-` 左对齐
- `:-:` 居中对齐
- `-:` 右对齐
