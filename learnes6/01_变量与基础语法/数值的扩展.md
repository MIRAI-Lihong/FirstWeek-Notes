# 数值的扩展

## 数值的表示

### 二进制和八进制表示法

- **二进制**：0b或0B
- **八进制**：0o或0O

```js
0b111110111 === 503 // true
0o767 === 503 // true
```

### 数值分隔符

较长的数值可以使用下划线（\_）作为分隔符，小数和科学计数法也可以用，增加**可读性**

> **使用注意点**
>
> 不能放在数值的最前面或最后面
> 不能两个或两个以上的分隔符连在一起
> 小数点的前后不能有分隔符
> 科学计数法里面，表示指数的e或E前后不能有分隔符
> Number()、parseInt()、parseFloat() 不支持数值分隔符

### Number上新增的API

#### Number.isFinite(), Number.isNaN()

- **Number.isFinite()**：检查一个数值是否为有限的，即不是`Infinity`
- **Number.isNaN()**: 检查一个值是否为NaN
- **isFinite()**: 将非数值的值转为数值，再检查是否为有限的
- **isNaN()**: 将非数值的值转为数值，再检查是否为NaN

```js
Number.isFinite(15) // true
Number.isFinite(NaN) // false
Number.isFinite(Infinity) // false
Number.isFinite(-Infinity) // false
Number.isFinite('15') // false

Number.isNaN(NaN) // true
Number.isNaN(true) // false
Number.isNaN(9 / NaN) // true

isFinite('25') // true
Number.isFinite('25') // false
isNaN('NaN') // true
Number.isNaN('NaN') // false
```

#### Number.parseInt(), Number.parseFloat()

```js
parseInt('12.34') // 12
parseFloat('123.45#') // 123.45

Number.parseInt('12.34') // 12
Number.parseFloat('123.45#') // 123.45
```

#### Number.isInteger()

判断一个数值是否为整数

```js
Number.isInteger(25) // true
Number.isInteger(25.1) // false
```

#### Number.EPSILON

JavaScript 能够表示的最小精度，值为2 的 -52 次方

```js
// 误差检查函数
function withinErrorMargin(left, right) {
  return Math.abs(left - right) < Number.EPSILON * Math.pow(2, 2)
}

0.1 + 0.2 === 0.3 // false
withinErrorMargin(0.1 + 0.2, 0.3) // true

1.1 + 1.3 === 2.4 // false
withinErrorMargin(1.1 + 1.3, 2.4) // true
```

#### Number.isSafeInteger()

安全整数：`Number.MAX_SAFE_INTEGER`(`Math.pow(2, 53) - 1`)和`Number.MIN_SAFE_INTEGER`(`-Number.MAX_SAFE_INTEGER`)

`Number.isSafeInteger()`：判断一个整数是否落在`Number.MAX_SAFE_INTEGER`和`Number.MIN_SAFE_INTEGER`之内

### Math上新增的API

- Math.trunc(num)：去除一个数的小数部分，返回整数部分。会进行类型转换
- Math.sign(num)：判断一个数到底是正数、负数、还是零。正数返回+1，负数返回-1，0返回0，-0返回-0。会进行类型转换
- Math.cbrt(num)：计算一个数的立方根。会进行类型转换
- Math.clz32(num)：将参数转为 32 位无符号整数的形式，然后返回这个 32 位值里面有多少个前导 0。会进行类型转换
- Math.imul(num1, num2)：返回两个数以 32 位带符号整数形式相乘的结果，返回的也是一个 32 位的带符号整数。
- Math.fround(num)：返回一个数的32位单精度浮点数形式。
- Math.hypot(num1, num2, ...)：返回所有参数的平方和的平方根。

---

- Math.f16round(num)：返回最接近输入值的16位半精度浮点数。超出±65,504 范围，返回 infinity
- Math.expm1(num)：返回 ex - 1，即Math.exp(x) - 1
- Math.log1p(num)：返回1 + x的自然对数，即Math.log(1 + x)。如果x小于-1，返回NaN
- Math.log10(num)：返回以 10 为底的x的对数。如果x小于 0，则返回 NaN
- Math.log2(num)：返回以 2 为底的x的对数。如果x小于 0，则返回 NaN

### BigInt

#### 简介

BigInt 只用来表示**整数**，没有位数的限制，任何位数的整数都可以精确表示

特点：

- 必须添加后缀n
- 可以使用各种进制表示
- `typeof 123n // 'bigint'`
- 可以使用负号，但是不能使用正号

#### 转换

- 使用 `BigInt(arg)` 生成 BigInt 类型的数值
- 使用`Boolean()、Number()`和`String()`这三个方法，将 BigInt 可以转为布尔值、数值和字符串类型
- 取反运算符（`!`）也可以将 BigInt 转为布尔值
