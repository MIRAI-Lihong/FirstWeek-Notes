# 字符串新增方法

## String.fromCodePoint()

- `fromCharCode`: 从 Unicode 码点返回对应字符，但是这个方法**不能识别**码点大于`0xFFFF`的字符

- `fromCodePoint`: 可以识别大于`0xFFFF`的字符，解决了`fromCharCode`的缺陷

```js
String.fromCharCode(0x20bb7) // ('ஷ')

String.fromCodePoint(0x20bb7) // ('𠮷')
```

## String.raw()

返回一个斜杠都**被转义**（即斜杠前面再加一个斜杠）的字符串

```js
String.raw`Hi\n${2 + 3}!` // ('Hi\\n5!')
```

## codePointAt()

处理 4 个字节储存的字符，返回一个字符的码点

```js
let s = '𠮷a'
for (let ch of s) {
  console.log(ch.codePointAt(0).toString(16))
}
// 20bb7
// 61
```

## includes(), startsWith(), endsWith()

- includes()：返回布尔值，表示是否找到了参数字符串
- startsWith()：返回布尔值，表示参数字符串是否在原字符串的**头部**
- endsWith()：返回布尔值，表示参数字符串是否在原字符串的**尾部**

```js
let s = 'Hello world!'

s.startsWith('world', 6) // true
s.endsWith('Hello', 5) // true
s.includes('Hello', 6) // false
```

## repeat(num)

返回一个新字符串，表示将原字符串重复num次

> num 为非 number 时，会进行转化，是小数会取整，是负数或者Infinity会报错

```js
'x'.repeat(3) // "xxx"
'hello'.repeat(2) // "hellohello"
'na'.repeat(0) // ""
```

## padStart(num, str)，padEnd(num, str)

主要用于**数值位数补全**

- `padStart()`用于头部补全
- `padEnd()`用于尾部补全

> 第一个参数是字符串补全生效的**最大长度**
> 第二个参数是用来**补全的字符串**，不传的话用**空格**补全
>
> 补全的字符串和原字符串长度之和**大于**了第一个参数，会截去超出位数的补全字符串

```js
'x'.padStart(5, 'ab') // 'ababx'
'x'.padStart(4, 'ab') // 'abax'

'x'.padEnd(5, 'ab') // 'xabab'
'x'.padEnd(4, 'ab') // 'xaba'
```

## trimStart()，trimEnd()

消除首尾**空格、tab建、换行符**等不可见空白符号。不会修改原字符串

- `trimStart()`消除字符串头部的空格
- `trimEnd()`消除尾部的空格

```js
const s = '  abc  '

s.trim() // "abc"
s.trimStart() // "abc  "
s.trimEnd() // "  abc"
```

## replaceAll(searchValue, replacement)

一次性替换所有匹配，返回一个新字符串，不会改变原字符串

> searchValue: 可以是一个字符串，也可以是一个全局的正则表达式（带有g修饰符,不然会报错）
>
> replacement：是一个字符串，表示替换的文本，其中可以使用一些特殊字符串
> `$&`：匹配的字符串。
> ``$` `` ：匹配结果前面的文本。
> `$'`：匹配结果后面的文本。
> `$n`：匹配成功的第n组内容，n是从1开始的自然数。这个参数生效的前提是，第一个参数必须是正则表达式。
> `$$`：指代美元符号$。

```js
'abbc'.replaceAll('b', '$&')
// 'abbc'
'abbc'.replaceAll('b', '$`')
// 'aaabc'
'abbc'.replaceAll('b', `$'`)
// 'abccc'
'abbc'.replaceAll(/(ab)(bc)/g, '$2$1')
// 'bcab'
'abc'.replaceAll('b', '$$')
// 'a$c'
```

## at(index)

返回指定位置的字符，支持负索引（倒数）

> 参数位置超出了字符串范围，返回 undefined

```js
const str = 'hello'
str.at(1) // "e"
str.at(-1) // "o"
```
