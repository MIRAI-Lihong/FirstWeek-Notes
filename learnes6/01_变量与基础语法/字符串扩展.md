# 字符串扩展

## 字符串表示方法

1. **普通转义**

```js
'\z' === 'z' // true
```

> `\` 表示转义字符

2. **八进制转义**

```js
'\172' === 'z' // true
```

> `\` 后面跟 1 到 3 位八进制数字（0-7）

3. **十六进制转义**

```js
'\x7A' === 'z' // true
```

> `\x` 后面跟 2 位十六进制数字（00-FF）

4. **Unicode转义**

```js
'\u007A' === 'z' // true
```

> `\u` 后面跟 4 位十六进制数字（0000-FFFF）

5. **Unicode码点转义（ES 6新增）**

```js
'\u{7A}' === 'z' // true
```

> `\u{}` 花括号内可以放 1 到 6 位十六进制数字

## 字符串遍历方式

采用 `for...of` 遍历，传统的for循环遍历识别字符可能出错

## JSON.stringify() 的改造

ES2019 改变了JSON.stringify()的行为，之前可以返回`0xD800`到`0xDFFF`之间的单个码点，但是 UTF-8 不允许这些码点单独使用，现在改为返回转义字符串。

> "\u{D834}" -> "\\u{D834}"

## 模板字符串

解决问题：**变量拼接字符的繁琐不方便**

既可以作为普通字符串，也可以作为多行字符串，还可以嵌入变量

```js
// 普通字符串
let str = `In JavaScript '\n' is a line-feed.`
// 多行字符串
`In JavaScript this is
 not legal.`

console.log(`string text line 1
string text line 2`)

// 字符串中嵌入变量 变量名写在${}中，里面也可以写表达式、调用函数
let name = 'Bob',
  time = 'today'
let res = `Hello ${name}, how are you ${time}?`
```

## 标签模板

```js
tag`Hello ${name}!`

// 等同于
tag(['Hello ', '!'], name)
```

- 第一个参数：一个数组，包含所有**非插值部分**的字符串片段
- 后续参数：模板中每个插值 ${} 的计算结果
